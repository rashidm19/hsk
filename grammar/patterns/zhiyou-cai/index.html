HSK 4 Pattern: 只有…才…

States a NECESSARY condition — B can ONLY happen if A is met. There is no other way to achieve B.

只有 + necessary condition, 才 + result

Examples / 例句

只有多练习，才能提高口语水平。
Zhǐyǒu duō liànxí, cái néng tígāo kǒuyǔ shuǐpíng.
Only by practicing more can you improve your speaking.
Practice is the ONLY way
只有亲自去一趟，才能了解真实情况。
Zhǐyǒu qīnzì qù yí tàng, cái néng liǎojiě zhēnshí qíngkuàng.
Only by going yourself can you understand the real situation.
No substitute for being there
只有大家一起努力，才能完成这个项目。
Zhǐyǒu dàjiā yìqǐ nǔlì, cái néng wánchéng zhège xiàngmù.
Only if everyone works together can we finish this project.
Collective effort is necessary
只有认真复习，才能在考试中取得好成绩。
Zhǐyǒu rènzhēn fùxí, cái néng zài kǎoshì zhōng qǔdé hǎo chéngjì.
Only by reviewing seriously can you get good grades on the exam.
认真 + 复习 + 取得 + 成绩 — all HSK 4. Classic study-advice frame in HSK 4 阅读
只有打好基础，将来才能学会更难的内容。
Zhǐyǒu dǎ hǎo jīchǔ, jiānglái cái néng xuéhuì gèng nán de nèiróng.
Only by laying a solid foundation can you later learn more difficult material.
将来 + 才能 = future result depends on present foundation; classic HSK 4 study advice

Common Errors / 常见错误

✗ 只有你努力，就能成功。
✓ 只有你努力，才能成功。/ 只要你努力，就能成功。
只有 pairs with 才 (not 就). If using 就, change to 只要.
✗ 只有今天天气好，才去爬山。
✓ 只要今天天气好，就去爬山。
Good weather is sufficient (not the ONLY factor) → 只要…就.
Easily confused: 只有…才 = necessary (ONLY way). 只要…就 = sufficient (one of many ways). Opposite logic! See comparison →

只有…才… FAQ

When is 只有…才… tested in HSK 4?

只有…才… (Only if... can...) appears in HSK 4 reading comprehension (Q66-85) when texts use complex sentences, in 阅读排序 (Q56-65) where this pattern's connectors signal sentence order, and in listening 段落 (Q26-45) where the speaker uses it to express states a necessary condition — b can only happen if a is met.

What's the most common mistake with 只有…才…?

只有…才 = necessary (ONLY way). 只要…就 = sufficient (one of many ways). Opposite logic!

Is 只有…才… required at HSK 4 or higher?
