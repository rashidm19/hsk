Scenario Dialogue / 情景对话

A student asks a courier clerk about sending a package. Read it aloud twice: once for meaning, once for fluency — the syllabus tests this task across listening and speaking.

A
您好，我想把这个包裹寄到上海，大概需要几天？
Nín hǎo, wǒ xiǎng bǎ zhège bāoguǒ jì dào Shànghǎi, dàgài xūyào jǐ tiān?
Hello, I would like to send this package to Shanghai. About how many days will it take?
B
普通快递三天左右，加急的明天就能到，不过要多付十五块。
Pǔtōng kuàidì sān tiān zuǒyòu, jiājí de míngtiān jiù néng dào, búguò yào duō fù shíwǔ kuài.
Standard delivery takes about three days; express arrives tomorrow, but costs fifteen yuan extra.
A
里面是一些易碎的盒子，麻烦您帮我包好一点儿。
Lǐmiàn shì yìxiē yìsuì de hézi, máfan nín bāng wǒ bāo hǎo yìdiǎnr.
There are some fragile boxes inside — could you please wrap it carefully?
B
没问题。请您先填一下这张快递单，写清楚收件人的地址和电话。
Méi wèntí. Qǐng nín xiān tián yíxià zhè zhāng kuàidì dān, xiě qīngchu shōujiànrén de dìzhǐ hé diànhuà.
No problem. Please fill in this delivery form first — write the recipient's address and phone number clearly.
A
好的，填完了。一共多少钱？
Hǎo de, tián wán le. Yígòng duōshao qián?
OK, done. How much altogether?
B
普通的二十二块。这是您的单号，可以随时查快递到哪儿了。
Pǔtōng de èrshí'èr kuài. Zhè shì nín de dānhào, kěyǐ suíshí chá kuàidì dào nǎr le.
Twenty-two yuan for standard. Here is your tracking number — you can check where the package is at any time.
